✅ Obrigado por participar da aula de hoje! Sua opinião faz a comunidade ADVBOX crescer. Avalie em menos de 1 minuto 👇 👉 [COLE AQUI O LINK DO FORMULÁRIO] São só 3 perguntas rápidas 🙏

💡 Dica: dispare a mensagem nos primeiros 10 minutos após a aula — o engajamento é maior enquanto o conteúdo ainda está fresco.

1
Abra script.google.com e crie um novo projeto. Cole o código do Apps Script (veja abaixo) e salve.
2
No menu, clique em Implantar → Nova implantação. Escolha tipo App da Web, acesso "Qualquer pessoa". Copie a URL gerada.
3
Neste arquivo HTML, localize a constante SHEET_URL e substitua pelo link copiado. Hospede o HTML no GitHub Pages ou em qualquer servidor estático.
4
Crie uma planilha Google com as colunas: Data/Hora · Nota NPS · Mais Útil · Melhoria · Aula/Turma. O script grava automaticamente cada resposta.
5
Encurte o link com bit.ly ou Linktree e coloque na mensagem do WhatsApp. Pronto! 🎉

Apps Script · cole no editor

function doPost(e) {
  const ss = SpreadsheetApp
    .openById('ID_DA_SUA_PLANILHA');
  const sh = ss.getActiveSheet();
  const d  = JSON.parse(e.postData.contents);
  sh.appendRow([
    new Date(),
    d.nota,
    d.util.join(', '),
    d.melhoria,
    d.aula || ''
  ]);
  return ContentService
    .createTextOutput(
      JSON.stringify({ok:true})
    )
    .setMimeType(
      ContentService.MimeType.JSON
    );
}
